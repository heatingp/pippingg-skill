# pippingg-skill

Claude Code skills for deep thinking and article writing.

## Installation

**Method 1: Add Marketplace**
```
/plugin marketplace add heatingp/pippingg-skill
```

**Method 2: Direct Install**
```
/plugin install thinking-skills@pippingg-skill
```

## Available Skills

### socratic-thinking

Socratic questioning partner for deep analysis. Helps you:

- Challenge assumptions with first-principles thinking
- Identify real problems vs emotional reactions
- Break out of mental frameworks
- Generate articles from conversations

**Usage:** `/socratic-thinking` or say "help me think", "I want to discuss"

### skill-iterator

Automatically iterate and update skills based on conversation feedback.

**Usage:** `/skill-iterator` or say "update this skill", "iterate on skill"
